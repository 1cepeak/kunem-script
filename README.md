# KunemScript

Утилита для перевода того что говорит Нарек Авагян в JavaScript

| Ключевое слово    | Перевод       |
|-------------------|---------------|
| `я не понимаю`    | `function`    |
| `понимаю`         | `return`      |
| `челик`           | `var`         |
| `микро челик`     | `const`       |
| `чел`             | `let`         |
| `твоя мать`       | `true`        |
| `твой батя`       | `false`       |
| `ты че конченный` | `if`          |
| `намана`          | `else`        |
| `да бля`          | `console.log` |
